Descartes

Styling for Programmers

An experimental library for writing CSS in JavaScript

Smart Stylesheets

Powered by JavaScript

I've taken some ideas from CSS preprocessors like Less and Sass and wondered what it would be like to write CSS as JSON. You can get all the advantages of a typical preprocessor and some: complex programming to set styles, access to the DOM, and even basic event listeners!

Click anywhere to see it in action

index.html 
<!doctype html>
<html>
  <head>
    <script type="text/javascript" src="descartes.js"></script>
  </head>
  <body>
    <h1>Hello World!</h1>
  <script type="text/javascript" src="styles.js"></script>
  </body>
</html>
styles.js 
var reset = {"margin": 0,
  "padding": 0,
  "box-sizing": "border-box"
};
var rand_angle = function() {
  return Math.round(Math.random() * (180) - 90);
};
var rand_rgba = function() {
  return "rgba("+[255,255,255].map(function(x) {
    return Math.round(Math.random() * x);
  }).join()+", 1)"
};

new Descartes({ // Just put descartes.js in the <head> tag
  "html": {
    "_mixins": reset, // Reuse CSS rules with mixins
    "body": { // Nest selectors just like in Sass and Less
      "_mixins": reset,
      "_listeners": [
        [window, "click"],
        [window, "touchstart"]], // Bind events...
      "background": function() {
        return 'linear-gradient('
          + rand_angle().toString() + 'deg,'
          + rand_rgba() + ','
          + rand_rgba() + ')'
      } // ...to set dynamic property values when they fire!
    }
  }
})

A Familiar Style

JavaScript like Less and Sass

It turns out writing CSS as JSON isn't all that different from Less and Sass. I've written Descartes to act just like other CSS processors with some extra features. You get nesting, mixins, and variables just like the others!

Learn the basics

Small, Fast, Easy

Just a small JavaScript file

Descartes is just under 30KB when minified. It's written entirely in JavaScript with its few dependencies baked in. It's also fast: the styles for this page took just milliseconds to compute and render.

 

Dynamic Properties

Pass functions as CSS values

You can set style properties as JavaScript functions that are accessed every time the Style Tree is rendered. This gives you access to the DOM, variables, and any of ther JavaScript you want!

 
 

Listeners

Bind events that updated your styles

You can also attach listeners to different DOM events that render a specific set of rules. That makes things like parallax, window based positioning, and even interactivity a breeze.

Mixins

Reuse your styles and functions

Reuse rules that you use for multiple elements by creating mixins. They get reapplied to each element you specify it for, letting you apply things like consistent styles, listeners, and even other mixins.
